1NF
The first normal form (1NF) states that every attribute 
type of a relation must be atomic and single-valued

2NF
A relation R is in second normal form (2NF) if it 
satisfies 1NF and every non-prime attribute type A in R 
is fully functionally dependent on any key of R.

3NF
A relation R is in third normal form (3NF) if it satisfies 
2NF and no non-prime attribute type of R is transitively 
dependent on the primary key.
A functional dependency 𝑋 → 𝑌 is a transitive 
dependency if there is a set of attribute types Z that is 
neither a candidate key nor a subset of any key of R, 
and both 𝑋 → 𝑍 and 𝑍 → 𝑌 hold
